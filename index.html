CALCULADORA




• La tecla ENTER muestra el resultado de la operación.
• La tecla r resetea los elementos que hay en la pantalla.
• Puedes escribir sin, cos o tan con el teclado para las operaciones trigonométricas.
• Puedes escribir pi con el teclado para obtener el parámetro π
• Puedes escribir ln con el teclado para obtener el parámetro ln2.
• Puedes escribir e con el teclado para obtener el parámetro e.
• Puedes escribir pot con el teclado para obtener la potencia.
• Puedes escribir ans con el teclado para volver al resultado anterior.
• Puedes eliminar el último digito marcando sup con el teclado.

• Si has escrito mal: cos, sin o tan, y no se escribe por pantalla, haz un reset.
• Solo se puede hacer una operacion de cos, sin o tan a la vez.
• Solo se puede hacer una operacion de √ a la vez.
• El parámetro ans solo funciona una vez realizado el reset.
